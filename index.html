통계 추적용 GitHub Pages 도메인 설정

GitHub Pages 활성화 후 부여받은 주소(예: https://username.github.io/repo-name)를 입력하세요. QR 코드가 해당 주소를 바라보게 자동 업데이트됩니다.

🔍 수집되는 구글 애널리틱스 주요 이벤트 목록

본 웹앱에는 모바일 스캔 시 구글 애널리틱스 4(GA4)로 다음과 같은 맞춤형 이벤트가 자동으로 보고되도록 내장되어 있습니다. 구글 애널리틱스 사이트의 [이벤트] 탭에서 바로 수치를 확인해 보세요.

audio_scan QR 코드를 스캔하여 오디오 재생 페이지에 진입할 때 관광지 현장의 실제 QR 코드 스캔 및 방문 횟수 집계
